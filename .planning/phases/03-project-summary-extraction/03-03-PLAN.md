   a. **`__init__(self, search_service: HybridSearchService, llm_service: GeminiService, citation_verifier: CitationVerifier)`**:
      - Store all three services as private attributes

   b. **`async def extract_project_summary(self, project_id: int) -> ProjectSummary`**:
      - Docstring: "Extract complete project summary from indexed documents using per-field retrieval and LLM extraction."
      - `start_time = time.time()`
      - `results: dict[str, ExtractedField] = {}`
      - Iterate over SUMMARY_FIELDS:
        - For each field_def in SUMMARY_FIELDS:
          1. **Retrieve**: Call `self._search_service.search(project_id=project_id, query=field_def.query, top_k=field_def.top_k, mode="hybrid")`
             - Store the returned `list[SearchResult]` as `chunks`
             - If no chunks returned, create an empty ExtractedField(value=None, confidence=0.0, confidence_level="low", requires_review=True) and continue to next field
          2. **Build context**: Call `build_labeled_context(chunks)` to get labeled context string
          3. **Build prompt**: Call `build_extraction_prompt(field_def, context)` to get the extraction prompt
          4. **Extract**: Call `await asyncio.to_thread(self._llm_service.extract, prompt=prompt, response_model=LLMExtractedField)` to get LLM extraction
             - Wrap in try/except: on any exception, log warning with field name and error, create empty ExtractedField with value=None and continue
          5. **Convert**: Convert LLMExtractedField to ExtractedField (copy value, confidence, citations, reasoning; set defaults for confidence_level="low", requires_review=True)
          6. **Verify**: Collect retrieval_scores = [chunk.score for chunk in chunks]. Call `self._citation_verifier.verify_field(field=extracted_field, source_chunks=chunks, retrieval_scores=retrieval_scores)`
          7. **Store**: `results[field_def.name] = verified_field`
          8. Log at INFO level: "Extracted {field_def.name}: value={'...' if value else 'None'}, confidence={confidence_level}"
        - Add a small delay between LLM calls to avoid rate limiting: `await asyncio.sleep(0.5)` between iterations (except after the last field). This is conservative -- can be reduced or removed if not hitting rate limits.
      - Build and return `ProjectSummary(**results)`

   c. **`async def extract_and_persist(self, project_id: int) -> ProjectSummary`**:
      - Docstring: "Extract project summary and persist to database."
      - Import inside method: `from app.database import async_session_factory`, `from app.models.project import Project`
      - Update project extraction_status to "in_progress":
        ```python
        async with async_session_factory() as session:
            project = await session.get(Project, project_id)
            if project is None:
                raise ValueError(f"Project {project_id} not found")
            project.extraction_status = "in_progress"
            await session.commit()
        ```
      - Try:
        - Call `summary = await self.extract_project_summary(project_id)`
        - Persist: update project with summary_json and extraction_status:
          ```python
          async with async_session_factory() as session:
              project = await session.get(Project, project_id)
              project.summary_json = summary.model_dump_json()
              project.extraction_status = "completed"
              await session.commit()
          ```
        - Return summary
      - Except Exception:
        - Log error
        - Update extraction_status to "failed":
          ```python
          async with async_session_factory() as session:
              project = await session.get(Project, project_id)
              if project:
                  project.extraction_status = "failed"
                  await session.commit()
          ```
        - Re-raise

   - Create lazy singleton for ExtractionService (same pattern as search API):
     ```python
     _extraction_service = None

     def _get_extraction_service():
         global _extraction_service
         if _extraction_service is None:
             settings = get_settings()
             if not settings.gemini_api_key:
                 raise HTTPException(
                     status_code=500,
                     detail="BIDOPS_GEMINI_API_KEY not configured. Set it in .env or environment."
                 )
             from app.services.indexing.embedding_service import EmbeddingService
             from app.services.search.hybrid_search import HybridSearchService
             from app.services.llm.gemini_service import GeminiService
             from app.services.extraction.citation_verifier import CitationVerifier
             from app.services.extraction.extraction_service import ExtractionService

             embedding_svc = EmbeddingService(
                 persist_dir=settings.chroma_persist_dir,
                 model_name=settings.embedding_model,
             )
             search_svc = HybridSearchService(embedding_service=embedding_svc)
             llm_svc = GeminiService(
                 api_key=settings.gemini_api_key,
                 model=settings.gemini_model,
             )
             verifier = CitationVerifier(
                 model_name=settings.nli_model,
                 confidence_high=settings.confidence_high_threshold,
                 confidence_low=settings.confidence_low_threshold,
                 review_threshold=settings.review_threshold,
             )
             _extraction_service = ExtractionService(
                 search_service=search_svc,
                 llm_service=llm_svc,
                 citation_verifier=verifier,
             )
         return _extraction_service
     ```
