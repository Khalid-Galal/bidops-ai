<objective>
Build the citation verification service using an NLI cross-encoder model that independently validates each citation produced by the LLM, and compute calibrated confidence scores.

Purpose: Combat citation hallucination (17-33% rate per Stanford research) by using an independent NLI model to verify that each cited source passage actually entails the extracted claim. This is the critical defense against the LLM generating plausible but incorrect citations. The verifier also computes combined confidence scores from three independent signals (retrieval, LLM, NLI) and flags low-confidence extractions for human review.

Output: CitationVerifier class providing verify_field() and calculate_confidence() methods.
</objective>

Create `app/services/extraction/citation_verifier.py`:

Import `from __future__ import annotations`, `logging`, `from sentence_transformers import CrossEncoder`, and use TYPE_CHECKING guard for `from app.schemas.extraction import Citation, ExtractedField` and `from app.services.search.hybrid_search import SearchResult`.

Implement `CitationVerifier` class:

1. **`__init__(self, model_name: str = "cross-encoder/nli-deberta-v3-xsmall", confidence_high: float = 0.8, confidence_low: float = 0.5, review_threshold: float = 0.5)`**:
   - Store model_name, thresholds
   - `_model: CrossEncoder | None = None` (lazy initialization)
   - Log the model name at INFO level

2. **`_get_model(self) -> CrossEncoder`** (private, lazy loader):
   - If `_model is None`:
     - Log "Loading NLI model: {model_name}" at INFO level
     - `self._model = CrossEncoder(self._model_name)`
     - Log "NLI model loaded" at INFO level
   - Return `self._model`

3. **`verify_citation(self, claim: str, source_text: str) -> float`**:
   - Docstring: "Check if source_text entails the claim. Returns entailment probability (0.0-1.0)."
   - Get model via `_get_model()`
   - Call `scores = model.predict([(source_text, claim)])`
   - The NLI model outputs scores for [contradiction, entailment, neutral] (3 classes)
   - Apply softmax to convert logits to probabilities:
     ```python
     import numpy as np
     logits = scores[0]  # shape: (3,)
     exp_logits = np.exp(logits - np.max(logits))  # numerical stability
     probs = exp_logits / exp_logits.sum()
     entailment_prob = float(probs[1])  # index 1 = entailment
     ```
   - Return `entailment_prob`
   - Handle errors: if model fails, log warning and return 0.0

4. **`_find_source_chunk(self, citation: Citation, source_chunks: list[SearchResult]) -> SearchResult | None`** (private):
   - Find the source chunk matching the citation's document_name and page_number
   - First try exact match on `filename == citation.document_name AND page_number == citation.page_number`
   - If no exact match, try matching just by `filename == citation.document_name` (LLM may cite wrong page)
   - If still no match, return None (citation references non-existent source)

5. **`verify_field(self, field: ExtractedField, source_chunks: list[SearchResult], retrieval_scores: list[float] | None = None) -> ExtractedField`**:
   - Docstring: "Verify all citations for an extracted field using NLI and compute confidence."
   - If `field.value is None` or empty string:
     - Set field.confidence = 0.0, field.confidence_level = "low", field.requires_review = True
     - Return field
   - If `field.citations` is empty:
     - Set field.confidence = max(0.1, field.confidence * 0.3)
     - Set field.confidence_level = "low", field.requires_review = True
     - Return field
   - For each citation in field.citations:
     - Find matching source chunk via `_find_source_chunk()`
     - If no source found: skip (don't include in verified list)
     - Build claim string: the citation's quote text
     - Call `verify_citation(claim=citation.quote, source_text=source.text)`
     - If entailment score >= 0.3 (lenient threshold for keeping citation -- the score contributes to overall confidence):
       - Keep citation, store its entailment score
     - Else:
       - Log at DEBUG: "Citation failed NLI verification (score={score}): {citation.document_name} p{citation.page_number}"
       - Remove from citations
   - Replace field.citations with verified citations only
   - Compute best retrieval score: `max(retrieval_scores) if retrieval_scores else 0.5`
   - Compute best NLI score: `max(entailment_scores) if entailment_scores else 0.0`
   - Call `calculate_confidence()` to get final score and level
   - Update field.confidence, field.confidence_level, field.requires_review
   - Return field

6. **`calculate_confidence(self, llm_confidence: float, retrieval_score: float, nli_entailment_score: float, has_verified_citation: bool) -> tuple[float, str, bool]`**:
   - Docstring: "Calculate combined confidence from three independent signals. Returns (score, level, requires_review)."
   - If `not has_verified_citation`:
     - score = min(0.3, llm_confidence * 0.3)
     - Return (score, "low", True)
   - Weighted combination (NLI most important, then retrieval, then LLM):
     - `score = nli_entailment_score * 0.5 + retrieval_score * 0.3 + llm_confidence * 0.2`
   - Determine level:
     - If score >= self._confidence_high: level = "high"
     - Elif score >= self._confidence_low: level = "medium"
     - Else: level = "low"
   - Determine requires_review: `score < self._review_threshold`
   - Return (score, level, requires_review)

**IMPORTANT NOTES:**
- Do NOT use the same LLM that generated the extraction to verify citations. This would be self-verification bias. The NLI cross-encoder is an INDEPENDENT model.
- sentence-transformers is already installed from Phase 2 (requirements.txt). The NLI model (~22MB) downloads automatically on first use.
- The CrossEncoder predict() output for nli-deberta-v3-xsmall returns raw logits, NOT probabilities. Must apply softmax.
- Import numpy (already available via sentence-transformers dependency) for softmax computation.
