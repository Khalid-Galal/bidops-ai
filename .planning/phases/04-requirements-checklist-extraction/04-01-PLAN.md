  <name>Task 1: Create checklist Pydantic schemas</name>
  <files>app/schemas/checklist.py</files>
  <action>
Create `app/schemas/checklist.py` with the following Pydantic models. Import `Citation` from `app.schemas.extraction` (reuse existing, do NOT duplicate).

1. **RequirementItem(BaseModel)** -- Schema passed to the LLM via instructor for extraction. Fields:
   - `requirement: str` -- Clear, concise statement of the requirement (Field with description)
   - `description: str = ""` -- Additional context or specific criteria
   - `is_mandatory: bool` -- True if uses mandatory language (shall, must, required). Field description should include guidance: "True for shall/must/required/mandatory, False for should/may/recommended"
   - `source_document: str` -- Filename of source document
   - `page_number: int` (ge=1) -- 1-based page number
   - `quote: str` (min_length=1) -- Exact verbatim quote from source
   - `confidence: float` (ge=0.0, le=1.0) -- LLM self-assessed confidence

2. **CategoryExtractionResponse(BaseModel)** -- Wrapper model for instructor extraction (NOT Iterable -- wrapper is more reliable with Gemini):
   - `items: list[RequirementItem] = Field(default_factory=list)` -- All requirements found for this category
   - `reasoning: str = ""` -- Brief summary of extraction approach

3. **VerifiedRequirement(BaseModel)** -- Post-NLI verified requirement:
   - `requirement: str`
   - `description: str = ""`
   - `category: str` -- Category name (e.g., "technical", "commercial")
   - `is_mandatory: bool`
   - `citation: Citation` -- Reused from app.schemas.extraction
   - `nli_score: float = Field(ge=0.0, le=1.0)` -- Raw NLI entailment score
   - `confidence: float = Field(ge=0.0, le=1.0)` -- Combined three-signal confidence
   - `confidence_level: str = "low"` -- "high", "medium", or "low"
   - `requires_review: bool = True`

4. **RequirementsChecklist(BaseModel)** -- Complete assembled checklist:
   - `requirements: list[VerifiedRequirement] = Field(default_factory=list)` -- Technical, commercial, legal, HSE requirements
   - `submission_documents: list[VerifiedRequirement] = Field(default_factory=list)` -- Mandatory docs
   - `eligibility_criteria: list[VerifiedRequirement] = Field(default_factory=list)` -- Pre-qual/eligibility
   - `total_count: int = 0`
   - `mandatory_count: int = 0`
   - `categories_extracted: list[str] = Field(default_factory=list)` -- Which categories were extracted

5. **ChecklistResponse(BaseModel)** -- API response wrapper:
   - `project_id: int`
   - `status: str` -- "completed", "in_progress", "failed", "not_started"
   - `checklist: RequirementsChecklist | None = None`
   - `extraction_time_seconds: float | None = None`
   - `total_requirements: int = 0`
   - `requirements_requiring_review: int = 0`

Use `from __future__ import annotations` at top. Add a module docstring explaining the schema hierarchy (same style as extraction.py). All Field() should have `description` parameters.
