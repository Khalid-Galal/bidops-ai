**1. Create `app/api/export.py`:**

   ```python
   """Export API endpoints for downloading project reports.

   Routes:
       GET /api/projects/{project_id}/export/excel - Download Excel report
       GET /api/projects/{project_id}/export/pdf   - Download PDF report
   """
   ```

   - Create router: `router = APIRouter(tags=["export"])`

   - `@router.get("/projects/{project_id}/export/excel")`:
     - Call `await generate_excel_report(project_id)` from excel_export module
     - Return `StreamingResponse(buffer, media_type="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet", headers={"Content-Disposition": f'attachment; filename="project_{project_id}_report.xlsx"'})`
     - Catch ValueError (project not found) -> HTTPException 404
     - Catch Exception -> HTTPException 500 with detail

   - `@router.get("/projects/{project_id}/export/pdf")`:
     - Call `await generate_pdf_report(project_id)` from pdf_export module
     - Return `StreamingResponse(buffer, media_type="application/pdf", headers={"Content-Disposition": f'attachment; filename="project_{project_id}_report.pdf"'})`
     - Catch ValueError -> 404
     - Catch RuntimeError (WeasyPrint not installed) -> HTTPException 501 with detail explaining how to install
     - Catch Exception -> 500

   Imports: `from fastapi import APIRouter, HTTPException, status`, `from fastapi.responses import StreamingResponse`, `from app.services.export.excel_export import generate_excel_report`, `from app.services.export.pdf_export import generate_pdf_report`.

**2. Update `app/main.py`:**
   - Add import: `from app.api.export import router as export_router`
   - Add router: `app.include_router(export_router, prefix="/api")` alongside existing router registrations (before the pages_router and static mount)
