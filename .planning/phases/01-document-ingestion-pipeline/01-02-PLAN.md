    2. Create `app/services/parsing/base.py` with the uniform parser output types (see research Pattern 2):

       ```python
       @dataclass
       class PageContent:
           page_number: int
           text: str
           tables: list[dict] = field(default_factory=list)
           # tables format: {"headers": [...], "data": [[...]], "rows": int, "cols": int}

       @dataclass
       class ParsedDocument:
           filename: str
           content_type: str           # "pdf", "docx", "xlsx"
           full_text: str              # Combined text for search/indexing (markdown format)
           pages: list[PageContent]    # Per-page/sheet breakdown
           tables: list[dict]          # All tables with page numbers
           metadata: dict              # Format-specific metadata (page_count, etc.)
           page_count: int
           processing_time_ms: int
           warnings: list[str] = field(default_factory=list)
       ```

       Create ParserInterface base class:
       ```python
       class ParserInterface:
           supported_extensions: list[str] = []

           def can_parse(self, filename: str) -> bool:
               ext = Path(filename).suffix.lower()
               return ext in self.supported_extensions

           async def parse(self, file_path: str) -> ParsedDocument:
               raise NotImplementedError
       ```

       Create `get_parser_for_file(filename: str) -> ParserInterface` function:
       - Maintain a list of parser instances: [PdfParser(), DocxParser(), XlsxParser()]
       - Loop through parsers, return first that can_parse(filename)
       - Raise ValueError if no parser supports the file extension

    3. Create `app/services/parsing/pdf_parser.py`:
       - Class PdfParser(ParserInterface) with supported_extensions = [".pdf"]
       - Create a module-level _converter variable (lazy initialization to avoid heavy import at startup)
       - `_get_converter()` function that creates and caches a DocumentConverter:
         - Import from docling.document_converter: DocumentConverter, PdfFormatOption
         - Import from docling.datamodel.pipeline_options: PdfPipelineOptions, EasyOcrOptions, TableStructureOptions
         - Import from docling.datamodel.base_models: InputFormat
         - Configure PdfPipelineOptions:
           - do_ocr=True
           - ocr_options=EasyOcrOptions(lang=["en"], use_gpu=False, force_full_page_ocr=False)
           - do_table_structure=True
           - table_structure_options=TableStructureOptions(do_cell_matching=True, mode="ACCURATE")
         - Create DocumentConverter with allowed_formats=[InputFormat.PDF] and format_options for PDF
       - async parse(self, file_path: str) -> ParsedDocument:
         - Record start time
         - Call _get_converter().convert(file_path) (this is sync -- Docling is not async)
         - Since Docling's convert() is CPU-bound and sync, wrap in asyncio.to_thread() to avoid blocking the event loop:
           `result = await asyncio.to_thread(converter.convert, file_path)`
         - Access result.document
         - Iterate items using doc.iterate_items():
           - For each item, check label via docling_core.types.doc.labels.DocItemLabel
           - Track page numbers from item.prov[0].page_no
           - For TABLE items: export to dataframe via item.export_to_dataframe(doc=doc) -- MUST pass doc= parameter (deprecated without it per research)
           - Build per-page text content
         - Get full_text via doc.export_to_markdown()
         - Get page_count from doc metadata or pages attribute
         - Build PageContent objects per page
         - Return ParsedDocument with all fields populated

       **Critical Docling notes from research:**
       - ALWAYS pass doc= to export_to_dataframe() (deprecated without it)
       - Wrap sync Docling convert() in asyncio.to_thread() to avoid blocking
       - Lazy-init the converter (first call downloads ~2GB of models)
       - Handle exceptions gracefully (return ParsedDocument with warnings list populated)

    4. Create `app/services/parsing/docx_parser.py`:
       - Class DocxParser(ParserInterface) with supported_extensions = [".docx"]
       - Share the Docling converter approach (Docling natively supports DOCX via InputFormat.DOCX)
       - Create a separate converter instance configured for DOCX:
         - allowed_formats=[InputFormat.DOCX]
         - No special pipeline options needed for DOCX (uses SimplePipeline)
       - async parse(self, file_path: str) -> ParsedDocument:
         - Same pattern as PDF parser but simpler (no OCR configuration)
         - Wrap convert() in asyncio.to_thread()
         - Extract text and tables from result.document
         - Build PageContent (DOCX pages are sections/pages from Docling's output)
         - Return ParsedDocument with content_type="docx"

    5. Create `app/services/parsing/__init__.py`:
       - Import and re-export: ParsedDocument, PageContent, ParserInterface, get_parser_for_file
       - Import all parser classes for registration

    **Error handling for all parsers:**
    - Catch exceptions during parsing, DO NOT crash
    - On error: return a ParsedDocument with empty content, the error in warnings list, and processing_time_ms set
    - Log the error via print() for v1 (structured logging deferred)

    Create `app/services/parsing/xlsx_parser.py` (see research XLSX Parsing code example):
    - Class XlsxParser(ParserInterface) with supported_extensions = [".xlsx", ".xls"]
    - async parse(self, file_path: str) -> ParsedDocument:
      - Record start time
      - Load workbook with openpyxl: `load_workbook(file_path, data_only=True, read_only=True)`
      - Wrap in asyncio.to_thread() since openpyxl is sync:
        `wb = await asyncio.to_thread(load_workbook, file_path, data_only=True, read_only=True)`
      - Iterate through wb.sheetnames:
        - For each sheet, iterate rows with sheet.iter_rows(values_only=True)
        - Skip completely empty rows (all cells None)
        - Convert cell values to strings (handle None -> "")
        - Build sheet text as pipe-delimited rows with sheet header ("=== Sheet: {name} ===")
        - Create table dict for each sheet with: {"page": sheet_idx+1, "sheet": sheet_name, "headers": first_row, "data": remaining_rows, "rows": count, "cols": count}
        - Create PageContent per sheet (each sheet = one "page"):
          - page_number = sheet_idx + 1
          - text = sheet text content
          - tables = [table_dict] if sheet has data
      - Close workbook: wb.close()
      - Build full_text by joining all sheet texts with double newline
      - Return ParsedDocument:
        - filename from Path(file_path).name
        - content_type = "xlsx"
        - full_text = combined text
        - pages = list of PageContent objects
        - tables = all table dicts across sheets
        - metadata = {"sheet_count": len(sheetnames), "sheet_names": list(sheetnames)}
        - page_count = number of sheets
        - processing_time_ms = elapsed time

    **Important:**
    - Use read_only=True to avoid loading entire workbook into memory (research pitfall 6)
    - Use data_only=True to get computed values instead of formulas
    - Handle the case where a sheet is completely empty (skip it, add warning)
    - Handle cells with dates/numbers by converting to str
    - Use try/except around the entire parse to catch corrupt files
