       - DocumentResponse: id, project_id, filename, file_type, file_size, status (str), page_count (int | None), processing_time_ms (int | None), error_message (str | None), created_at (datetime)

       - UploadResponse: task_id (str), uploaded (int), skipped (int), filenames (list[str])
       - ProgressEvent: status (str), total (int), processed (int), current_file (str), errors (list[dict])

    4. Create `app/api/health.py`:
       - GET /health returns {"status": "ok", "version": "0.1.0"}

    5. Create `app/api/projects.py` with APIRouter(prefix="/projects", tags=["projects"]):
       - POST /projects -- create project, accepts ProjectCreate body, returns ProjectResponse (201)
       - GET /projects -- list all projects, returns list[ProjectResponse], ordered by created_at desc
       - GET /projects/{project_id} -- get single project by id, returns ProjectResponse, raises 404 if not found
       - DELETE /projects/{project_id} -- delete project and its documents, returns 204
       - All endpoints use Depends(get_db) for database session
       - Use `select()` syntax (SQLAlchemy 2.0 style), NOT legacy query API
       - For create: db.add(project), await db.commit(), await db.refresh(project)
       - For list: result = await db.execute(select(Project).order_by(Project.created_at.desc())), projects = result.scalars().all()
       - For get: project = await db.get(Project, project_id), if None raise HTTPException(404)
       - For delete: await db.delete(project), await db.commit()

    6. Create `app/api/__init__.py` (empty).

    7. Update `app/main.py` to include both routers:
       - app.include_router(health_router, prefix="/api")
       - app.include_router(projects_router, prefix="/api")

    **Important:** Use async def for all endpoint functions. Use await for all database operations.
